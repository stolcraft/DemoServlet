0- Dans le dossier avec le src et le POM,
lancer une commande cmd et écrivez la phrase
"mvn clean package"

1- Deplacer le fichier DemoServlet.war dans le dossier target et mettez le 
dans votre serveur.

2- (Pour tomcat )Executer startup.bat situé dans le dossier bin de tomcat

3- Dans l'url du navigateur internet, entrer la commande suivante : http://localhost:8080/DemoServlet/

4- Dans l'interface Web, cliquez sur le bouton "infos servlet" et l'ensemble des informations 
seront inscrite dans la console du serveur.
